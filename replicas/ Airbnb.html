Airbnb (today’s MVP) — Founder Kit

Short-stay listings: host onboarding, search + filters, bookings, payouts, reviews.

Generate this in demo Get early access

Overview

Goal: Validate a supply + demand marketplace in one region with high-quality listings and safe, reliable bookings.

  • Primary users: Guests, Hosts, Ops Admin
  • Platforms: Web App (guest + host), lightweight Admin panel
  • Payments: Stripe Connect (host payouts), escrowed charges, refunds
  • Risk controls: Email/phone verification, ID/KYC stub, simple fraud checks

MVP Scope (must-have)

  • Auth: email + magic link (or OAuth stub)
  • Guest: search, filters (price, dates, guests), listing details, calendar, booking + payment
  • Host: onboarding, create/edit listing, photos upload, pricing, availability calendar, payouts
  • Reviews: guest→host and host→guest (post-stay)
  • Admin: view users/listings/bookings, resolve disputes, refunds

Core Screens

  1. Home/search: location + date picker, results grid with map preview
  2. Listing details: gallery, amenities, host profile, calendar, reviews, “Reserve” CTA
  3. Checkout: price breakdown, fees, payment capture, confirmation
  4. Host dashboard: listings table, create/edit flow, availability, payouts
  5. Admin read-only panel: users, listings, bookings with basic actions
