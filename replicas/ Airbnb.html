Airbnb (today’s MVP) — Founder Kit

Short-stay marketplace: host onboarding, search + filters, bookings, payouts, reviews.

Overview

Goal: Validate a short-stay marketplace with host onboarding, guest booking, and reviews.

  • Primary users: Hosts, Guests, Admin
  • Platforms: Web app (host + guest), Admin panel
  • Payments: Stripe (Connect Express) for payouts
  • Risk controls: ID verification, reviews, dispute management

MVP Scope (must-have)

  • Host onboarding
  • Property listings with photos, pricing, and availability
  • Search + filters (location, price, dates)
  • Booking and payments
  • Guest reviews

Future Features (nice-to-have)

  • Host dashboards with analytics
  • Loyalty or referral program
  • Advanced pricing tools (dynamic pricing)
