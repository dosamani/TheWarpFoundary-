Luvable (today’s MVP) — Founder Kit

D2C baby-care brand: storefront, PDPs, checkout, subscriptions, UGC reviews.

Get early access

Overview

  • Users: Shoppers, Marketing, Ops
  • Platform: Web storefront (mobile-first), lightweight admin
  • Catalog: Diapers, wipes, creams, bundles
  • Monetization: One-time + subscriptions

MVP Scope (must-have)

  • Home + collections + product detail pages (PDPs)
  • Cart & checkout (Stripe or Shopify Checkout)
  • Subscriptions (monthly bundles) & order history
  • UGC reviews (text + rating) with light moderation
  • CMS blocks for promos, hero banners, FAQs, policies

Core Screens

  1. Home: hero, featured products, social proof
  2. Collection: filters (size, price, category)
  3. PDP: gallery, variant/size selector, subs toggle, reviews
  4. Cart/Checkout: address, shipping, payments, confirmation
  5. Account: subscriptions, orders, address book

Data Model (simplified)

User{id, email, name}
Product{id, title, desc, images[], variants[], price}
Review{id, productId, userId, rating, text, createdAt}
Order{id, userId, items[], total, status}
Subscription{id, userId, productId|bundleId, interval, status}

API (stub)

GET  /api/products?collection=bundles
GET  /api/products/:id
POST /api/cart/add { productId, variantId, qty, subscribe }
POST /api/checkout/create { cartId }
GET  /api/orders/:id
POST /api/reviews { productId, rating, text }

Go-to-Market (0–6 weeks)

  • Influencer seeding (new parent creators) + UGC
  • Bundles with free first-month shipping
  • Email flows: abandoned cart, reorder reminder, milestone coupons

Risks & Mitigations

  • Retention: subscription discounts, flexible skips
  • Trust: dermatology claims → clear labeling & reviews
  • Ops: track OOS + backorder messaging

Next Actions

  • Ship PDP + checkout demo with a starter bundle
  • Add 20 real reviews (seeded/testimonials)
  • Run a 100-customer beta with subscription incentives
